CRITICAL Issue #1 — CORS missing Capacitor origins
ALLOWED_ORIGINS in api/index.ts did not include capacitor://localhost or http://localhost. Every API call from the Android/iOS native app would be CORS-blocked.
✅ Fixed — commit d6c591b
Added capacitor://localhost, http://localhost, https://localhost to ALLOWED_ORIGINS. Verified live: access-control-allow-origin: capacitor://localhost
CRITICAL Issue #2 — Cloud vault uses relative URLs on native
cloud-vault-sync.ts used relative /api/vaults/cloud fetch paths. On Capacitor native (origin: capacitor://localhost), these resolve to the local WebView server — not the production API. All cloud vault sync operations would silently fail on device.
✅ Fixed — commit d6c591b
Added const CLOUD_API = 'https://www.ironvault.app' and replaced all 6 relative fetch paths with absolute URLs.
